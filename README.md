# CHATIFY 
Chatify is a Flutter/Firebase based messaging app in active development.
It implements the main features of a chat app:\
\
***Authentication***\
***Profile***\
***Sending, receiving, updating and deleting messages***

```

$ cd Chatify
$ flutter pub get
$ flutter run
```
